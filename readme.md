# Hostel Issue Management System

## Project Overview
This is a web-based Hostel Issue Management System built as a hackathon
project to streamline the reporting and resolution of hostel maintenance
issues.

Students can report issues, and hostel management can view, track,
prioritize, and resolve them through an admin dashboard.

## Features
- Role-based authentication (Student & Management)
- Issue reporting with duplicate detection
- Admin dashboard with issue statistics
- Issue filtering by status and category
- Secure backend APIs with JWT authentication

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

### Frontend
- HTML
- CSS
- Vanilla JavaScript

## How to Run Locally
1. Clone the repository
2. Navigate to the backend folder
3. Install dependencies:
   npm install
4. Create a `.env` file using `.env.example`
5. Start the server:
   npm start
6. Open frontend HTML files in a browser
